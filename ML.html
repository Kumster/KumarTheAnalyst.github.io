This machine learning project predicts housing prices using structured data from a Kaggle dataset. The goal is to estimate final sale prices based on features like living area, neighborhood, quality, and more.

The notebook covers end-to-end ML pipeline steps including:

  • Data cleaning & preprocessing
  • Feature engineering and encoding
  • Model selection (Ridge, Lasso, Random Forest, Gradient Boosting)
  • Performance evaluation using RMSE

The project was built using Python, scikit-learn, Pandas, and Seaborn. Final evaluation was performed on a holdout set, and the best-performing model achieved a Root Mean Squared Error of approximately 27,897.
