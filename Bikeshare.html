This machine learning project classifies whether bike pickup demand (PU) exceeds dropoff demand (DO) using hourly bikeshare data and weather indicators. It simulates real-world allocation strategies in public bike systems.

The notebook covers end-to-end supervised classification steps including:

  • Feature engineering using weather and temporal data
  • Dummy encoding categorical values (weather icons)
  • Target engineering (binary label PU > DO)
  • Training and visualizing classification trees
  • Evaluating models: SVC, Decision Tree, KNN
  • 5-fold Cross-Validation using AUC scores
  • Reporting final out-of-sample model performance

This project was built using Pandas, scikit-learn, Matplotlib, and Capital Bikeshare + Weather datasets.

The final model selected was Linear SVM, achieving an out-of-sample AUC of approximately 38.75%.
